🚀 Portfolio de Projets : Systèmes, Réseaux & IA
Bienvenue dans ce dépôt regroupant mes travaux académiques et techniques. Ce projet est divisé en trois axes majeurs : Gestion de parc (ITSM), Intelligence Artificielle et Infrastructure Réseau.

🛠️ 1. Gestion d'Actifs & Helpdesk (GLPI)
Mise en place d'une solution complète de gestion des services informatiques (ITSM) pour automatiser la maintenance et le support.

Objectif : Centraliser l'inventaire et simuler un centre de services professionnel.

Stack Technique : Linux, GLPI, FusionInventory.

Points Clés :

Installation et durcissement d'un serveur GLPI sous Linux.

Automatisation : Déploiement d'agents FusionInventory pour la remontée automatique du matériel et des logiciels.

Helpdesk : Configuration des workflows de gestion d'incidents (Tickets, notifications).


🤖 2. Algorithmes d'IA & Recherche (Python)
Implémentation de concepts fondamentaux de l'Intelligence Artificielle pour comprendre les mécanismes d'apprentissage et d'optimisation.

Projets inclus :

Perceptron Simple : Développement de l'algorithme de base en Python pour appréhender les fondements des réseaux de neurones (classification binaire).

Algorithme A (Pathfinding) :* Implémentation d'un moteur de recherche de chemin optimal pour la résolution de problèmes de navigation.

Stack Technique : Python, NumPy.


🌐 3. Infrastructure Réseau Sécurisée (Cisco)
Conception et sécurisation d'architectures réseau complexes pour garantir la disponibilité et la confidentialité des données.

Logiciel utilisé : Cisco Packet Tracer.

Réalisations :

Segmentation : Création de VLANs par services et configuration du routage inter-VLAN (Router-on-a-stick).

Sécurité Niveau 2 : Mise en place de Port Security pour limiter les accès physiques.

Filtrage : Implémentation d'ACLs (Access Control Lists) pour restreindre les flux entre les zones sensibles.
